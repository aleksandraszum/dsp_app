Sygnały można opisać za pomocą parametrów. Najczęściej są to: 
  • wartość średnia sygnału
  • energia sygnału
  • moc sygnału

Parametry sygnału ciągłego:

Wartość średnia sygnału

x ¯ = 1 t 2 - t 1 ∫ t 1 t 2 x ( t ) d t ,

gdzie:
x ¯ - wartość średnia sygnału x ,
[ t 1 , t 2 ] - przedział czasu trwania sygnału.
W przypadku sygnałów o nieskończonym czasie trwania, średnią wyznacza się ze wzoru:
x ¯ = lim T → ∞ 1 2 T ∫ - T T x ( t ) d t .
Jeżeli analizowany sygnał nieskończony, jest sygnałem okresowym o okresie T 0 , uśrednianie w czasie nieskończonym jest równoważne uśrednianiu sygnału za okres:
x ¯ = 1 T 0 ∫ t 0 t 0 + T 0 x ( t ) d t .
Przy czym t 0 jest chwilą dowolną.

Moc sygnału

Moc sygnału ciągłego P x definiowana jest wzorem:
P x = lim T → ∞ 1 2 T ∫ - T T x ( t ) 2 d t .
W sytuacji, jeżeli analizowany sygnał jest sygnałem okresowym, moc można wyznaczyć z zależności:
P x = 1 T 0 ∫ t 0 t 0 + T 0 x ( t ) 2 d t ,
gdzie:
T 0  jest okresem, a t 0 dowolną chwilą czasu.


Wzory, które definiują podstawowe parametry sygnału dyskretnego są podobne do parametrów sygnału ciągłego z tą różnicą, że całkę zastąpiono sumą.

Parametry sygnału dyskretnego:

Wartość średnia sygnału

Wartość średnia sygnału dyskretnego x [ n ] wyraża się wzorami:
  • x ¯ = 1 n 2 - n 1 + 1 ∑ n = n 1 n 2 x ( n )
dla sygnału x ( n ) określonego na skończonym przedziale [ n 1 , n 2 ] .
  • x ¯ = lim N → ∞ 1 2 N + 1 ∑ n = - N N x ( n )
dla sygnału o nieskończonym czasie trwania.
  • x ¯ = 1 N ∑ n = n 0 n 0 + N 0 - 1 x ( n )
dla sygnału okresowego o okresie N 0 . n 0 dowolną liczbą całkowitą.

Moc sygnału

Moc sygnału dyskretnego P x definiuje się wzorem:
P x = lim N → ∞ 1 2 N + 1 ∑ - N N x 2 ( n ) .
W przypadku, gdy sygnał jest okresowy, moc można wyznaczyć korzystając z zależności:
P x = 1 N 0 ∑ n 0 n 0 + N 0 - 1 x 2 ( n ) ,
gdzie N 0 jest okresem, a n 0 dowolną liczbą całkowitą.
