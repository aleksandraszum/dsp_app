Transformatę Fouriera sygnału ciągłego s ^ ( f ) bądź sygnału dyskretnego S [ n ] nazywa się widmem sygnału lub gęstością widmową. Widmo sygnału ma aspekt czysto matematyczny.

Widmo amplitudowe oraz widmo fazowe

Widmo amplitudowe oraz widmo fazowe są rzeczywistymi charakterystykami sygnału, które opisują gęstość amplitudy oraz gęstość fazy sygnału.


Amplitudę sygnału definiuje się następująco:
  • dla całkowej transformaty Fouriera: A ( f ) = | s ^ ( f ) | = s r e 2 ^ ( f ) + s i m 2 ^ ( f )
  • dla dyskretnej transformaty Fouriera: A [ n ] = | s ^ ( [ n ] | = s r e 2 ^ [ n ] + s i m 2 ^ [ n ]

Fazę sygnału definiuje się następująco:
  • dla całkowej transformaty Fouriera:  ϕ ( f ) = arctan s i m ^ ( f ) s r e ^ ( f )
  • dla dyskretnej transformaty Fouriera:  ϕ [ n ] = arctan s i m ^ [ n ] s r e ^ [ n ]
