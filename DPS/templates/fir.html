Filtry FIR są to filtry o skończonej odpowiedzi impulsowej. Związane jest to z faktem, że transmitancja filtrów FIR jest wielomianem zmiennej zespolonej z, wskutek czego odpowiedź impulsowa filtrów FIR jest skończona, a liczba niezerowych próbek odpowiedzi jest równa rzędowi filtru. Filtry FIR nie posiadają biegunów, co powoduje, że filtry FIR są stabilne.

Stosunkowo nietrudno jest zaprojektować tak filtr FIR, aby jego charakterystyka fazowa była liniowa. Warunkiem, który zapewni liniową charakterystykę fazową, jest odpowiedź impulsowa postaci: h ( n ) = h ( N - 1 - n )
pamiętając, że transmitancja filtru ma postać:
H ( z ) = ∑ n = 0 N - 1 h ( n ) z - n

Filtr FIR można zaprojektować wykorzystując metodę okien czasowych. Filtry są okienkowane funkcją sinc. Tego rodzaju filtry wykorzystuje się w celu oddzielenia jednego pasma częstotliwości od drugiego. Cechują się stabilnością oraz osiągnięciem parametrów o wysokim poziomie w dziedzinie częstotliwości kosztem słabego działania w dziedzinie czasu.

Odpowiedź impulsowa idealnego filtrów dolno-  h d ( t ) i górnoprzepustowego h g ( t ) wynoszą:
h d ( t ) = sin ( 2 π f c t ) π t ,   dla   t   ≠ 0   2 f c ,   dla   t   =   0 h g ( t ) = -   sin ( 2 π f c t ) π t ,   dla   t   ≠ 0   1   -   2 f c ,   dla   t   =   0
Warto zauważyć, że dla t = 0 jest inny wzór. Związane jest to z faktem, że w przypadku zastosowania drugiego wzoru nastąpiłoby dzielenie przez 0. Dokonanie splotu analizowanego sygnału z jedną z powyższych odpowiedzi impulsowych utworzy filtr dolno- lub górnoprzepustowy.

Biorąc pod uwagę, że uzyskany filtr jest tylko przybliżeniem filtru idealnego, uzyskana odpowiedź częstotliwościowa również nie jest idealna. Aby poradzić sobie z tą niedogodnością, stosuje się tzn. okna. Polega to na tym, że wymnaża się odpowiedź impulsową przez wybrane przez siebie okno. Działanie to usuwa gwałtowne opadanie na obciętych krańcach odpowiedzi impulsowej i w efekcie końcowym poprawia uzyskaną odpowiedź częstotliwościową.
