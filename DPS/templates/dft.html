Ze względu na odmienny charakter sygnałów ciągłych i dyskretnych, przekształcenia Fouriera będą definiowane w inny sposób dla sygnałów dyskretnych.

Dyskretne przekształcenie Fouriera

Przekształcenie Fouriera dla sygnału s [ k ] = s [ 0 ] , s [ 1 ] , … , s [ N - 1 ] definiuje się następująco:
S [ N ] = ∑ k = 0 N - 1 s [ k ] e - j 2 π n k / N ,
gdzie n ∈ < 0 ; N - 1 > .
Funkcję S [ N ] nosi nazwę widma sygnału dyskretnego s [ k ] .

Widmo sygnału dyskretnego  S [ N ] posiada tę własność, że jest okresowe. Okres widma jest równy częstotliwości próbkowania sygnału.

Odwrotne dyskretne przekształcenie Fouriera

Odwrotne przekształcenie Fouriera umożliwia przekształcenie dyskretnego sygnału zdefiniowanego w dziedzinie częstotliwości na dyskretny sygnał zdefiniowany w dziedzinie czasu. Przekształcenie to można zdefiniować następująco:
s [ k ] = 1 N ∑ n = 0 N - 1 S [ n ] e j 2 π k n / N ,
gdzie k ∈ < 0 , N - 1 > .

Właściwości dyskretnego przekształcenia Fouriera

Założenia: x [ k ] , y [ k ] są sygnałami zespolonymi, natomiast X ( n ) , Y ( n ) są transformatami sygnałów x [ k ] , y [ k ] .
  • liniowość: a x [ k ] + b y [ k ] ↔ a X ( n ) + b Y ( n )
  • przesunięcie w czasie: x [ k - k 0 ] ↔ X ( n ) e - k 0
  • przesunięcie w dziedzinie częstotliwości.
