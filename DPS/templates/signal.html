δ   ( t ) = 0 dla t   ≠   i 1 dla t   =   i

Skok jednostkowy
Sygnał nazywany skokiem jednostkowym jest odpowiedzią układu na wymuszenie w postaci skoku jednostkowego przy zerowych warunkach początkowych. Można go zdefiniować wzorem:

Charakterystyka skokowa, odpowiedź skokowa – w teorii sterowania, jedna z charakterystyk czasowych, która wraz z charakterystyką impulsową oraz charakterystykami częstotliwościowymi stanowi podstawowy opis działania układu regulacji. Charakterystyka skokowa to odpowiedź układu na wymuszenie w postaci skoku jednostkowego przy zerowych warunkach początkowych. Dla wymuszenia w postaci skoku jednostkowego:
1 ( t ) = 0 dla t < 0 1 dla t ≥ 0

Sygnał sinusoidalny
Sygnał sinusoidalny jest sygnałem zdefiniowanym następująco:
s ( t ) = A sin ( ω t ) = A sin ( 2 π f t )

Parametrami sygnału sinusoidalnego są:
  1. amplituda
  2. wartość międzyszczytowa
  3. wartość skuteczna
  4. okres sygnału
