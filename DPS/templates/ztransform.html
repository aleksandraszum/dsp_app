Aby móc projektować filtry cyfrowe, należy wykorzystać przekształcenie Ƶ . Przekształcenie Ƶ polega na przebadaniu odpowiedzi impulsowej sinusoidami i przebiegami wykładniczymi w celu znalezienia zer i biegunów.

Transformatą Ƶ  sygnału x [ n ] , definiuje się wzorem:
X ( z ) = ∑ n = - ∞ ∞ x [ n ] z - n ,
gdzie z ∈ ℂ.
Transformata Ƶ jest odpowiednikiem przekształcenia Laplace'a dla funkcji ciągłych.
Transformata istnieje tylko wówczas, gdy szereg jest zbieżny. Zbieżność zachodzi, kiedy obszar zmienności wartości z analizowanego sygnału wynosi: 0   ≤   r -   <   | z |   < r +   ≤ ∞ .

Transmitancja sygnału

Transmitancją sygnału H ( z ) , czyli funkcją przenoszenia systemu, nazywa się stosunek transformaty sygnału wyjściowego Y ( z ) do transformaty sygnału wejściowego X ( z ), czyli:
H ( z ) = Y ( z ) X ( z ) .
Z transmitancji sygnału można wyznaczyć zależność: Y ( z ) = H ( z ) X ( z ) , która opisuje zależność między sygnałem wejściowym, a sygnałem wyjściowym.

Transmitancję sygnału wykorzystuje się do projektowania filtrów cyfrowych.
