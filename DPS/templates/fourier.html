Szeregiem Fouriera nazywa się szereg, który umożliwia rozłożenie funkcji okresowej na sumę funkcji trygonometrycznych. Aby uzyskać szereg Fouriera, należy spełnić warunki Dirichleta.

Warunki Dirichleta

Założenia:
  • funkcja f jest okresowa o okresie równym T .
Warunki:
  • funkcja f spełnia warunek: ∫ - T 2 T 2 | f ( t ) | d t < ∞ .
  • funkcja f w przedziale jednego okresu posiada skończoną liczbę ekstremów lokalnych.
  • funkcja f w przedziale jednego okresu posiada skończoną liczbę punktów nieciągłości pierwszego rodzaju.

Wzór funkcji f po rozwinięciu jej w szereg Fouriera przybiera postać:
f ( t ) = a 0 2 + ∑ k = 1 ∞ ( a k cos ( k π t T ) + b k sin ( k π t T ) ) ,
gdzie a 0 , a k , b k to stałe zdefiniowane jako:
a 0 = 1 T ∫ - T T f ( t ) d t , a k = 1 T ∫ - T T f ( t ) cos ( k π t T ) d t , b k = 1 T ∫ - T T f ( t ) sin ( k π t T ) d t .
Funkcję f można także rozwinąć w zespolony szereg Fouriera:
f ( t ) = ∑ k = - ∞ ∞ c k exp ( j k π t T )
gdzie j to jednostka urojona, natomiast c k to współczynnik zdefiniowany następująco:
c k = 1 2 T ∫ - T T f ( t ) exp ( - j k π t T ) d t
